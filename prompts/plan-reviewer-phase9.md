# Plan-Reviewer (Codex) — Phase 9: PRD Review

Review the Planner's PRD (`prd/PRD.md`) and technical design (`prd/tech-design.md`).

## Check

1. **Clarity**: Does the PRD clearly state what skill to build and why?
2. **Scope**: Are diagnostic scenarios well-defined? At least 3?
3. **Design**: Does tech-design specify SKILL.md format, target path, validation?
4. **Feasibility**: Can the Developer actually execute this? Are all paths correct?
5. **Constraints**: Are constraints realistic (read-only for other agents)?

```
verdict: PASS | REQUEST_CHANGES
summary: <1-line>
findings:
  - <specific issue if REQUEST_CHANGES>
```
